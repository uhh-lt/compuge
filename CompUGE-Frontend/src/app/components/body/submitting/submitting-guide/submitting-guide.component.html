How to Submit a Model

Please fill out the form above with the required details. The following fields are optional but strongly recommended:

  • Team Name
  • Contact Email
  • Model Link (This could be a download link, a Hugging Face model hub link, a GitHub link, or any other link that provides access to your model.)

Ensure the uploaded predictions file is a CSV with a column named "predictions", containing the model's predictions in the same order as the test data.

If you opt to make your predictions private, the model link will also be kept private.
